Módulos

sys:

Este módulo permite o acesso a algumas variáveis utilizadas e mantidas pelo interpretador e a funções que interagem frequentemente com o interpretador.

Documentação - Python Software Fundation

getopt:

Este módulo ajuda scripts a fazer o parse dos argumentos da linha de comando armazenados em sys.argv. Suporta as mesmas convenções do Linux getopt().

Documentação - Python Software Fundation

os:

Este módulo fornece uma via portátil de usar as funcionalidades dependentes do sistema operativo.

Se pretender:

  • Escrever e ler ficheiros, consulte open()
  • Manipular paths, consulte os.path
  • Ler as linhas de um ficheiro, consulte fileinput
  • Criar ficheiros temporários e directorias, consulte tempfile
  • Fazer uma gestão de alto-nível de ficheiros e directorias, veja shutil
Documentação - Python Software Fundation

fileinput:

Este módulo implementa funções que facilitam a escrita de um loop para o parse do standard input ou de uma lista de ficheiros.

Documentação - Python Software Fundation

shutil:

O módulo shutil oferece operações de alto-nível em ficheiros e colecções de ficheiros. Em particular, são incluídas funções que suportam a cópia e remoção de ficheiros.

Documentação - Python Software Fundation

subprocess:

O módulo subprocess permite criar novos processos, conectar os respectivos pipes de input/output/erro e obter os códigos de retorno.

Este módulo tem o intuito de substituir alguns módulos antigos e funções:

  • os.system
  • os.spawn
Documentação - Python Software Fundation

pandas:

O módulo pandas oferece estruturas desenhadas para fazer com que o trabalho que envolva dados relacionais ou etiquetáveis seja fácil e intuítivo.

Documentação - pydata.org
Pandas Cheat Sheet

matplotlib:

O módulo matplotlib permite o output de gráficos numa grande variedade de formatos e ambientes interactivos.

Documentação - matplotlib.org

ntlk:

O NLTK (Natural Language Toolkit) é um conjunto de bibliotecas e programas para processamento de liguagens naturais (NLP).

Documentação - nltk.org

pprint:

O módulo pprint fornece a capacidade de fazer pretty-print de estrututras de dados do Python.

Documentação - Python Software Fundation

shelve:

O módulo shelve funciona como um dicionário de dados persistente.

Documentação - Python Software Fundation
